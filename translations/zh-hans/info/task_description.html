在这项任务中，您必须制作Minimum spanning tree (Wikipedia) 。

您会得到一个顶点坐标列表。每个坐标都是 x（整数）和 y（整数）的元组。
您必须返回一个连接所有顶点的线段列表（或可迭代线段）。 线条的总长度应为最小值。每条线都是由两个整数组成的元组。 这些整数代表输入列表的索引。

注意

  • 测试的输出具有唯一的行组合。

示例

{% if interpreter.slug == "js-node" %} 

{% else %}

connect_stars([(1, 1), (4, 4)]) == [(0, 1)]                         # or [(1, 0)]
connect_stars([(1, 1), (4, 1), (4, 4)]) == [(0, 1), (1, 2)]         # or [(1, 2), (0, 1)] , [(1, 0), (2,1)] etc
connect_stars([(6, 6), (6, 8), (8, 4), (3, 2)]) == [(0, 1), (0, 2), (0, 3)]   # or [(2, 0), (0, 3), (1, 0)] etc


{% endif %}
