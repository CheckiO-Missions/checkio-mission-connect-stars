In dieser Mission müssen Sie die Minimum spanning tree (Wikipedia) erstellen.

Sie erhalten eine Liste mit Koordinaten von Eckpunkten. Jede Koordinate ist ein Tupel aus x (Ganzzahl) und y (Ganzzahl).
Sie müssen eine Liste (oder eine Iterable) von Linien zurückgeben, die alle Eckpunkte verbinden. Die Gesamtlänge der Linien sollte das Minimum sein. Jede Linie ist ein Tupel aus zwei ganzen Zahlen. Diese Ganzzahlen stellen den Index der Liste der Eingaben dar.

HINWEIS:

  • Die Ausgabe von Tests hat eine eindeutige Kombination von Zeilen.
