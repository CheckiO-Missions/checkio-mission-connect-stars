Dans cette mission, vous devrez réaliser le site Minimum spanning tree (Wikipedia).

On vous donne une liste de coordonnées de sommets. Chaque coordonnée est un tuple de x (entier) et y (entier).
Vous devez retourner une liste (ou un itérable) de lignes qui relient tous les sommets. La longueur totale des lignes doit être minimale. Chaque ligne est un tuple de deux entiers. Ces entiers représentent l'index de la liste d'entrée.

REMARQUE:

  • La sortie des tests a une combinaison unique de lignes.
