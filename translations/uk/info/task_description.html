У цій місії вам потрібно буде зробити Minimum spanning tree (Wikipedia).

Вам задано список координат вершин. Кожна координата є кортежем з x (ціле число) та y (ціле число).
Ви повинні повернути список (або ітерабельний список) прямих, які з'єднують усі вершини. Сумарна довжина ліній повинна бути мінімальною. Кожен рядок є кортежем з двох цілих чисел. Ці числа є індексами списку вхідних даних.

ЗАУВАЖЕННЯ:

  • Вихідні дані тестів мають унікальну комбінацію рядків.
